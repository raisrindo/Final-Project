Deskripsi

Web ini adalah Final Project dalam kegiatan JavaScript Development Course yang saya ikuti.
Berisi Exercise 1, 2, dan 3 dan Final Poject ini ini sendiri :

1. Exercise 2
Buatlah sebuah halaman HTML yang berisi tentang data diri kalian. Gunakan tag-tag HTML untuk membuat halaman tersebut dan gunakan CSS untuk mempercantik tampilannya!! Jangan lupa untuk upload halaman tersebut ke Github dengan nama repository "Exercise 1".

2. Exercise 2
Buatlah sebuah page yang berfungsi sebagai kalkulator. Buatlah semenarik mungkin!! Jangan lupa untuk menggunakan CSS dan upload halaman tersebut ke Github dengan nama repository "Exercise 2".

3. Exercise 3
Buatlah sebuah halaman dengan kriteria sebagai berikut: Buatlah table dengan data dari Swapi Tambah field untuk filtering data dari table tersebut Upload ke Github dengan nama repository "Exercise 3"

4. Final Project
Untuk final project kalian, buatlah sebuah web dengan kriteria sebagai berikut: Memiliki slider image Menggunakan form dan gunakanlah Formspree untuk melihat hasil submit dari form tersebut Menggunakan modal Memiliki 3 halaman (minimal) Upload ke Github dengan nama repository "Final Project"
